🧩 Проблема

Создание чат-ботов и интерактивных сценариев до сих пор требует программистов. Low-code конструкторы решают часть задач, но:

  • Требуют ежемесячной подписки и хостинга у поставщика
  • Не поддерживают автономный запуск и локальное хранение данных
  • Используют собственные визуальные DSL, непонятные ИИ
  • Плохо интегрируются с внешним кодом и кастомной логикой

🧩 The Problem

Creating chatbots and interactive scenarios still requires programmers. Low-code builders solve some problems, but:

  • Require monthly subscriptions and vendor hosting
  • Don't support autonomous deployment and local data storage
  • Use proprietary visual DSLs that are unclear to AI
  • Poor integration with external code and custom logic

💡 Решение

BotTable AI — self-hosted low-code/ИИ-платформа, где таблица становится кодом.

  • Каждая строка описывает переход состояния или действие
  • CSV/XLSX-файл заменяет сложные скрипты и JSON-конфиги
  • Встроенный интерпретатор выполняет таблицу как FSM
  • ИИ-модуль генерирует таблицы по текстовому описанию

💡 The Solution

BotTable AI — self-hosted low-code/AI platform where tables become code.

  • Each row describes a state transition or action
  • CSV/XLSX files replace complex scripts and JSON configs
  • Built-in interpreter executes tables as FSM
  • AI module generates tables from text descriptions
